## Tech Stack / Tools
- **Game Engine:** Unity 6 (6000.0.60f1 LTS)
- **AI Navigation:** NavMesh (pathfinding)
- **Level Design:** ProBuilder
- **Animations:** Animator Controller (state machine for enemy states)
- **3D Assets:** Unity Asset Store & Adobe Mixamo (imported models/animations)
